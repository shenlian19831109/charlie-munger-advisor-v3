# Charlie Munger Knowledge Graph v3.1

Mind Distiller Method v3.1 标准化知识图谱

## 📊 结构

- `knowledge-graph/`: 完整的网状知识图谱
  - `entities.json`: 实体定义
  - `relationships.json`: 关系定义（四维度量化）
  - `simple_graph.json`: 简化版（core+important）
  - `visualization-d3-interactive.html`: 交互式可视化

## 🎯 四维度量化

- **权重** (weight): 0.5-1.0
- **重要性** (importance): core/important/reference
- **方向** (direction): forward/backward/bidirectional
- **时间** (start_year/end_year): 重要事件
