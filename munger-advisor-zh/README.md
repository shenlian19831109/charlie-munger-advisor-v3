# Charlie Munger Thinking Assistant

Think with Charlie Munger's logic and perspective to analyze investment, business, and life decisions.

---

## Project Overview

**Charlie Munger** (1924-2023) was Vice Chairman of Berkshire Hathaway and Warren Buffett's 60+ year partner, hailed as "the smartest investor."

This skill distills Munger's core thinking system:
- Latticework of Mental Models
- Psychology of Human Misjudgment (25 cognitive biases)
- Investment Philosophy (Economic Moat, Circle of Competence, Margin of Safety)
- Life Wisdom (Inversion, Long-term thinking, Rational decision-making)

## Core Framework

### Munger's Three Pillars

| Pillar | Core Question |
|--------|--------------|
| Inversion | "Where would I die?" |
| Multi-Model | Cross-disciplinary latticework |
| Long-term | Secret to compounding is living long enough |

## Usage

### Dialogue Example

```
You: Analyze my investment using Munger's framework

Munger:
- First: Is this a good business?
- Then: Is management rational and honest?
- Then: Is the price reasonable?
- Then: Can I understand this business?
- Then: Where would I die?
- Finally: Is this my best opportunity?
```
